# Side Meetings

This project displays a calendar view for all side meetings bookings from Cal.

## Update Info for New Meeting

Update the `meeting.json` file with the new info and build + deploy.

## Setup

Requirements: **Node.js 24.x**

1. Create a copy of `.env.sample` to `.env` and set the values.
2. Install npm dependencies:
   ```sh
   npm install
   ```

### Run in dev mode

```sh
npm run dev
```

### Build for production

```sh
npm run build
```
